# Souza Lucena & Advogados Associados

Site institucional do escritório **Souza Lucena & Advogados Associados**, desenvolvido em HTML, CSS e JavaScript puro, hospedado via GitHub Pages.

🌐 **[souzalucena.com.br](https://souzalucena.com.br)**

---

## 📋 Sobre o Projeto

Site institucional de escritório de advocacia sediado em Recife/PE, com foco em apresentação profissional, captação de clientes e conformidade com as normas de publicidade da OAB.

---

## ⚙️ Tecnologias

- **HTML5** semântico
- **CSS3** com variáveis customizadas, Grid e Flexbox
- **JavaScript** vanilla — sem frameworks
- **Google Fonts** — Cormorant Garamond + DM Sans
- **Schema.org** — structured data para SEO (LegalService)
- **Open Graph** — preview em redes sociais e WhatsApp

---

## 🎨 Design

- Paleta: creme, dourado (`#b8915a`) e carmesim (`#7a1c18`)
- Tipografia: Cormorant Garamond (títulos) + DM Sans (corpo)
- Animações de scroll reveal via IntersectionObserver
- Layout responsivo (mobile-first)
- Contador animado nos indicadores da seção Sobre

---

## 📦 Funcionalidades

- ✅ Navbar fixa com scroll e menu mobile (drawer)
- ✅ Botão WhatsApp flutuante com mensagem pré-definida
- ✅ Links diretos para WhatsApp, Instagram e agendamento online
- ✅ Banner de cookies (LGPD)
- ✅ Google Maps integrado
- ✅ Sitemap XML + robots.txt para SEO
- ✅ Páginas legais com visual consistente (Política de Privacidade, Termos de Uso, Aviso Legal)

---

## 🚀 Deploy

O site é hospedado via **GitHub Pages** com domínio customizado `souzalucena.com.br`.

Para atualizar o site, basta fazer commit das alterações na branch `main` — o deploy é automático.
