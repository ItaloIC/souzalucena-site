Política de Privacidade

Última atualização: 2026

O escritório Souza Lucena & Advogados Associados respeita a privacidade dos usuários e se compromete a proteger os dados pessoais coletados através deste site, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD).

1. Coleta de Informações

Podemos coletar informações quando você acessa nosso site, como:

  • Endereço IP
  • Navegador utilizado
  • Páginas acessadas
  • Tempo de navegação
  • Informações enviadas voluntariamente pelo usuário

2. Uso das Informações

As informações coletadas podem ser utilizadas para:

  • Melhorar a experiência do usuário no site
  • Analisar estatísticas de acesso
  • Responder contatos realizados pelo usuário
  • Cumprir obrigações legais

3. Uso de Cookies

Este site pode utilizar cookies para melhorar a experiência de navegação e analisar o tráfego de visitantes.

O usuário pode desativar os cookies diretamente nas configurações do seu navegador.

4. Compartilhamento de Dados

O escritório não vende nem compartilha dados pessoais com terceiros, exceto quando necessário para cumprimento de obrigações legais.

5. Segurança das Informações

Adotamos medidas técnicas e administrativas adequadas para proteger os dados contra acesso não autorizado ou uso indevido.

6. Direitos do Usuário

Nos termos da LGPD, o usuário possui direito de solicitar:

  • Acesso aos seus dados pessoais
  • Correção de dados incompletos ou incorretos
  • Exclusão de dados pessoais
  • Revogação do consentimento

7. Alterações desta Política

Esta política poderá ser atualizada periodicamente.

8. Contato

Em caso de dúvidas sobre esta política, entre em contato através dos canais disponíveis neste site.
